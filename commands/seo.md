Add performance optimizations and SEO improvements to `$ARGUMENTS`.

## Tasks

1. **Image Optimization**
   - Optimize images for fast loading
   - Use proper sizing and modern formats (WebP, AVIF)
   - Implement lazy loading where appropriate

2. **Meta Tags & SEO**
   - Add proper meta title and description
   - Implement Open Graph tags for social sharing
   - Add canonical URLs where needed

3. **Structured Data**
   - Add JSON-LD structured data markup
   - Include relevant schema types (Organization, Product, Article, etc.)
   - Validate with Google's Rich Results Test

4. **Core Web Vitals**
   - Optimize Largest Contentful Paint (LCP)
   - Minimize Cumulative Layout Shift (CLS)
   - Improve First Input Delay (FID) / Interaction to Next Paint (INP)

## Output

- Updated code with optimizations applied
- Summary of SEO improvements made
- Recommendations for further optimization
